Para que serve o git ?

O git é um sistema de controle de versão de códigos.
Gerencia ramificações feitas por diferentes pessoas de um time de modo a manter a ordem e integração de todas as partes.
Ou seja, mesmo que várias pessoas estejam trabalhando em um programa ao mesmo tempo, ainda é possível controlar as mudanças para evitar erros. {Data-28-07-2024-03:27}
